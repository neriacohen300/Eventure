What it does

Everything you need
for a stunning slideshow

From raw photos to polished export — Eventure handles every step with intelligent processing and a clean UI.

🖼️
Smart Image Processing
Auto-fits images to 1920×1080 with blurred background fill, EXIF rotation correction, and fast parallel processing.
✏️
Text Overlays with RTL
Add custom text to any slide with full Hebrew / RTL bidirectional support and customizable fonts.
🎬
Adobe Premiere Export
Exports a ready-to-use Premiere Pro project folder with pre-processed images and transitions — just add effects.
📊
PowerPoint Import
Import .pptx files, extract slides as images and text, and convert them into slideshow format automatically.
🎵
Audio Support
Attach audio files to your slideshow with duration management, syncing music to the length of your presentation.
🌍
Multi-Language UI
Switch between English and Hebrew interfaces. Theme system for full visual customization.
Workflow

From folder to finished
in a few clicks

01
Import your photos
Point Eventure at a folder of images — JPEG, PNG, WebP, and even GIFs are all supported. The app will load them with EXIF orientation data intact.
Drag & Drop
02
Arrange & customise
Reorder slides, set durations, pick transitions, add text overlays, and attach background music. Crop and rotate each photo individually.
PyQt5 UI
03
Process in parallel
Eventure uses multiprocessing to resize, compose, and blur all images to 1920×1080 simultaneously — no waiting around.
Multiprocessing
04
Export your way
Export directly to video, save a .slideshow file, or generate an Adobe Premiere Pro project folder with a preset pack ready to apply your effects.
Premiere / Video
Get started

Install from source

Requires Python 3.10+ on Windows. Or just grab the pre-built setup.exe from the Releases page.

1 — Clone the repo
git clone https://github.com/neriacohen300/Eventure.git cd Eventure
2 — Install dependencies
pip install -r requirements.txt
3 — Run the app
python Eventure.py

Key dependencies: PyQt5 · Pillow · python-pptx · openpyxl · python-bidi
