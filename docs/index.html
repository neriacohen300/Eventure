What it does

Everything you need
for a stunning slideshow

From raw photos to polished export — Eventure handles every step with intelligent image processing and a clean UI.

🖼️
Smart Image Processing
Auto-fits photos to 1920×1080 with blurred background fill, EXIF rotation correction, crop support, and multiprocessing speed.
✏️
Text Overlays with RTL
Add custom text to any slide with full Hebrew / RTL bidirectional text support and customizable font families.
🎬
Adobe Premiere Export
Generates a ready-to-use Premiere Pro project folder. Add your favourite effects preset and you're done.
📊
PowerPoint Import
Import .pptx files, extract slides as images and text, and convert them into slideshow format automatically.
🎵
Audio Support
Attach audio files with duration management that syncs music length perfectly to your slideshow.
🌍
Multi-Language UI
Switch between English and Hebrew interfaces, with a full theme system for visual customization.
Workflow

From folder to finished
in a few clicks

01
Import your photos
Point Eventure at a folder of images — JPEG, PNG, WebP, and GIFs are all supported. EXIF orientation is applied automatically.
Drag & Drop
02
Arrange & customise
Reorder slides, set durations, pick transitions, add text overlays, and attach background music. Crop and rotate each photo individually.
PyQt5 UI
03
Process in parallel
Eventure uses multiprocessing to resize, compose, and blur all images to 1920×1080 simultaneously — no waiting around.
Multiprocessing
04
Export your way
Export to video, save a .slideshow project file, or generate an Adobe Premiere Pro folder with images pre-processed and ready for effects.

Get started

Install from source

Requires Python 3.10+ on Windows. Or grab the pre-built setup.exe — no Python needed.

1 — Clone the repo
git clone https://github.com/neriacohen300/Eventure.git cd Eventure
2 — Install dependencies
pip install -r requirements.txt
3 — Run
python Eventure.py

Key dependencies: PyQt5 · Pillow · python-pptx · openpyxl · python-bidi
